IMPORTANT: The actual sqlite JDBC driver JAR is NOT included here due to environment restrictions.
Please download the correct JAR and place it in this lib/ folder with the name sqlite-jdbc.jar

Recommended download URL (official Maven Central):
https://repo1.maven.org/maven2/org/xerial/sqlite-jdbc/3.45.2.0/sqlite-jdbc-3.45.2.0.jar

After downloading:
1. Rename the file to sqlite-jdbc.jar
2. Move it into the project's lib/ folder
3. Restart VS Code and run: Java: Clean Java Language Server Workspace
